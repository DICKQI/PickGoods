# 功能特性详细说明

本文档描述当前前端已经实现的页面、交互和主要能力，以路由和组件实现为准。

## 云展柜（Cloud Showcase）

**路由**：`/showcase`

云展柜是首页主入口，顶部通过 Tab 切换三块能力：

| Tab | 功能 | 主要组件 |
|-----|------|----------|
| 展柜 | 我的展柜 / 公共展柜切换、展柜内容编排 | `ShowcaseManager` |
| 谷仓 | 谷子搜索、筛选、分页、详情、排序 | `SearchBar` + `FilterPanel` + `GoodsCard` + `GoodsDrawer` |
| 统计看板 | 谷子资产统计与可视化 | `StatsDashboard` |

### 展柜

展柜用于管理个人或公开展示页：

- 支持创建、编辑、删除展柜
- 支持上传展柜封面
- 支持在「我的展柜」与「公共展柜」之间切换
- 支持向展柜添加、移除、上下移动谷子
- 支持展柜内谷子右键菜单操作
- 展柜列表使用前四张谷子主图做预览

> 当前前端不提供“展柜内二级分类”功能，展柜仅按顺序管理谷子条目。

### 谷仓

谷仓是收藏资产检索与列表视图：

| 功能 | 说明 |
|------|------|
| 智能搜索 | `300ms` 防抖搜索 |
| 多维筛选 | IP、角色、品类、主题、状态、是否官谷、位置、分组显示 |
| 双视图 | 标准视图 / 相似随机视图 |
| 响应式布局 | PC 网格列表，移动端卡片列表 |
| 详情抽屉 | 点击卡片查看详情与图片 |
| 右键菜单 | 编辑、删除、前移、后移、置顶本页顶部 |
| 跨页排序 | 支持前一页 / 后一页锚点排序 |

### 统计看板

统计看板展示当前筛选条件下的资产概览与分布：

- 谷子件数、总数量、估算总金额
- 状态分布
- 官谷 / 同人分布
- 作品类型分布
- IP TopN
- 品类 TopN

当前页面主要展示概览和分布图，趋势字段已在 API 类型中预留，但界面未做趋势图表。

### 通用交互

- 顶部 Tab 和页面标题会随路由与当前模块变化
- 云展柜右下角有悬浮刷新按钮
- 仅在「谷仓」Tab 显示新增谷子按钮
- 移动端会显示底部导航栏

## 统计看板（Stats Dashboard）

**位置**：`/showcase` 页面内的「统计看板」Tab

| 功能 | 说明 |
|------|------|
| 统计筛选 | Top N、官谷/同人、状态、IP、品类、入手日期、录入日期 |
| 概览卡片 | 件数、总数量、估算总金额 |
| 图表 | 状态分布、官谷/同人、作品类型、IP TopN、品类 TopN |
| 自动刷新 | 筛选变化后自动重算 |

## 登录与认证

**路由**：`/login`

- 支持登录和注册
- 登录后 Token 会写入本地存储
- 路由守卫会拦截未登录访问
- 设置页可查看当前用户信息并退出登录

---

## 草稿箱（Goods Drafts）

**路由**：`/goods/drafts`

- 仅显示状态为 `draft` 的谷子
- 支持分页
- 点击条目可继续编辑
- 提供返回和新增谷子入口

---

## 资产录入（Goods Form）

**路由**：`/goods/new`、`/goods/:id/edit`

完整的谷子新增和编辑表单，支持：

- IP、角色、品类、主题联动
- 多角色关联
- 主图上传
- 上传前裁剪
- 原生相机拍照
- 补充图片上传、删除、标签管理
- 重复检测与合并提示
- 草稿保存与继续编辑

---

## 位置管理（Location Management）

**路由**：`/location`

- 树形位置导航
- 位置详情
- 位置下谷子列表
- 位置 CRUD
- 位置路径高亮跳转

---

## IP 与角色管理（IP & Character Management）

**路由**：`/ipcharacter`

**兼容路由**：`/ip`、`/character`

- IP 列表与角色列表统一管理
- IP 关键词
- 角色头像上传
- 按 IP 展开/折叠角色列表
- IP 拖拽排序
- BGM 搜索作品并导入角色

### BGM 导入流程

1. 搜索 Bangumi 作品
2. 获取作品角色
3. 选择并批量导入
4. 查看导入结果

---

**路由**：`/category`

- 品类 CRUD
- 父子层级管理
- 拖拽排序

## 主题管理（Theme Management）

**路由**：`/theme`

## 设置（Settings）

**路由**：`/settings`

- 运行时修改后端 API 地址
- 查看当前后端地址与默认地址
- 当前账号信息
- 退出登录
- 管理员可进入管理后台

## 管理后台（Admin）

**路由**：`/admin/*`

- 用户管理
- 谷子管理
- IP 与角色管理
- 品类管理
- 主题管理

---

## 移动端优化

项目针对移动端做了专门适配：

- 底部导航栏固定展示
- 云展柜、位置管理、IP 与角色、品类、主题五个主入口
- 谷子详情抽屉支持移动端底部弹出
- 部分页面支持安全区域适配
- 图片裁剪、原生相机等移动端能力已接入 Capacitor
